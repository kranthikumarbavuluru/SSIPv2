# MeitY standalone preview applications

The MeitY preview launchers from v3.4.3.8.0.4 through v3.4.3.8.0.8 are
developer diagnostics only after v3.4.3.8.1.

Normal users use the public dashboard on port 8502.
Administrators use the Admin Review application on port 8505.

Do not delete the diagnostic files yet. They remain useful for regression
testing and detailed forensic inspection.
