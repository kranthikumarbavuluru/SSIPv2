# SSIP v3.4.3.7.9 — DST Department Page Alignment

This phase aligns the DST department page with the successful MeitY
department-page structure.

## DST page structure

- DST Schemes
- Current DST Calls
- DST Historical Archive

The page hero and metric cards show permanent programmes, open calls,
upcoming calls and qualified historical calls.

## Historical archive consolidation

The existing governed DST historical archive remains intact. Its four
relevance categories are all visible and must reconcile exactly to the
qualified archive total:

- Startup Relevant
- Startup Ecosystem Call
- Review Required
- General DST

The existing chart, filters, pagination, evidence cards and official links
are retained. Historical records never display an active Apply action.

## Live Calls page

The detailed DST archive is removed from the general Live Calls page to
avoid duplication. Live Calls retains only:

- Open & Current
- Closed Startup Calls

It provides direct links to the DST and MeitY department archives.

## Governance

- No database changes.
- No publication changes.
- No call or programme identities are renamed.
- No archive rows are removed.
- No current/historical status is inferred.
