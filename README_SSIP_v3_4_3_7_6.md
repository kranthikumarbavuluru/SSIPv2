# SSIP v3.4.3.7.6 — Three-Column Admin Review Workspace

This UI-only phase makes scheme and call verification easier on a wide
desktop screen.

## Reading layout

The selected record remains visible in a sticky header. The review area
shows three columns at the same time:

1. identity, organization and parent relationship;
2. official evidence, links, dates and application status;
3. readiness checks, blockers, duplicates, warnings and audit history.

## Editing layout

The existing editable fields are no longer hidden behind review tabs.
They are displayed in three columns:

- identity, organization and relationship;
- status, dates and official evidence;
- funding and support.

Structured content fields are also spread across three columns.

## Governance

The phase does not alter:

- review decisions;
- approved, rejected or pending records;
- staging or publication status;
- database schema;
- approval validation;
- duplicate protection;
- audit history.
